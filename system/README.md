# Solar System [C++ OpenGl]

## Required Software 
 * **Sublime Text [Optional]**
 * **C++**

## Required Dependencies
 * ScalaFX
 * OpenGL
 * OpenCV

## Images
 [github link](https://github.com/Sectah/Solar-system) 

## Compile
 * cd folder
 * $ g++ -std=c++11 -pthread main.cpp -lGL -lGLU -lGLEW -lglut `pkg-config --cflags --libs opencv`
